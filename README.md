# Iris-Flower-Project
Machine Learning project for classifying Iris flower species using Python and K-Nearest Neighbors (KNN).

## Project Overview

This project uses Machine Learning to classify Iris flowers into three different species based on their physical measurements.

The classification is performed using the K-Nearest Neighbors (KNN) algorithm. The project includes data exploration, visualization, model training, evaluation, and prediction of flower species.

## Dataset Information

The dataset contains 150 flower samples belonging to three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

### Features

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

### Target Variable

* Species

## Machine Learning Model

Algorithm Used:

**K-Nearest Neighbors (KNN)**

The model was trained to classify iris flowers based on their measurements and achieved excellent classification accuracy.

## Key Findings

* Iris Setosa is clearly distinguishable from the other two species.
* Petal Length and Petal Width are highly correlated features.
* Petal measurements are the most important features for classification.
* The KNN model achieved high prediction accuracy.
* Machine Learning can effectively classify flower species using flower measurements.

## Conclusion

This project successfully demonstrates the application of Machine Learning for classification tasks. Through data preprocessing, visualization, and model training, the KNN algorithm accurately classified iris flower species based on their characteristics.
